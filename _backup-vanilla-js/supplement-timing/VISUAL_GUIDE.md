## Animation & Transitions

### Card Hover Effect
```
Normal State:
  transform: translateX(0px)
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1)

Hover State:
  transform: translateX(5px)
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.15)

Transition: all 0.3s ease
```

### Card Fade In
```
@keyframes fadeIn {
  from {
    opacity: 0
    transform: translateY(20px)
  }
  to {
    opacity: 1
    transform: translateY(0)
  }
}

Duration: 0.3s
Easing: ease-out
```

### Button Hover
```
Normal State:
  background: #FF6B35
  transform: translateY(0)

Hover State:
  background: #FF6B35 (same)
  transform: translateY(-2px)
  box-shadow: deeper shadow

Active State:
  transform: translateY(0)
```

## Responsive Breakpoints

### 1400px+ (Desktop)
- 3-4 column summary grid
- Wide timeline (max-width: 1400px)
- Full sidebar support
- Large cards

### 768px - 1400px (Tablet)
- 2 column summary grid
- Medium timeline
- Slightly smaller cards

### 480px - 768px (Mobile)
- 1 column summary grid
- Full width timeline
- Compact cards
- Stacked buttons

### < 480px (Small Mobile)
- Extra compact layout
- Smaller fonts
- Tighter spacing
- Minimal padding
